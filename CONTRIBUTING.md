# Contributing

If you want to contribute, you must speak french!

Si vous souhaitez contribuer, vous devez parler français !

## Overall

Suivez [cette convention](https://gist.github.com/anhgelus/4ffbd42abcf427d3bbc2919d835374b9).

Ensuite, si possible, signez vos commits, merci :3

## Versioning

Ici, on suit les conventions, et donc le [semver](https://semver.org/) !

## Style de code

Merci de suivre les conventions qu'on adopte ainsi que la manière avec laquelle on développe.

Toutes merges requests ne les respectant pas se verront refuser.
